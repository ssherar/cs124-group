## Feature List

### Main Feature List

* Create a new blank document
* Automatic resize/scroll of canvas
* Add a class rectangle
	* Move
	* Resize
	* Delete
	* Add and delete methods/attributes
	* Automatic resizing
* Labelling
	* Moving/creating/deleting
	* Add on relationships
	* Add on Canvas
* Relationship
	* Add/Delete
	* Modify type
	* Add cardinalities
	* Change direction
* Management
	* Save
	* Load
	* Export
		* Image
		* Code
	* Print
	* Undo/Redo
	* Right Click Functionality
